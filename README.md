# SI.NET-IoT
 
## Greenhouse

This app is really simple, just 4 types of sensor at 30 places in one awesome greenhouse.
App components:
- Gui *[.Net Core 5 Razor Pages]*
- API *[.Net Core 5]*
- Publisher *[Python]*
- Queue *[RabbitMQ]*
- Database *[MongoDB]*
